## Getting Started

### Run the app
```bash
flutter run
```

### Run on emulator
```bash
flutter emulators --launch Medium_Phone_API_36.1
flutter run -d emulator-5554
```

### Hot reload
Press `r` in the terminal to see changes instantly.

## Requirements
- Flutter SDK 3.41.0+
- Dart 3.11.0+
- Android Studio (for emulator)
